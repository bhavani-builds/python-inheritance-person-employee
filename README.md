# 👨‍💼 Person and Employee Management System (Python OOP)

A simple Python project demonstrating **Inheritance** in Object-Oriented Programming (OOP). This project creates a base class `Person` and a derived class `Employee` that inherits the properties of the `Person` class.

---

## 📌 Features

- Create a `Person` class with name and age
- Create an `Employee` class that inherits from `Person`
- Add an employee ID
- Display employee details
- Demonstrates inheritance using the `super()` function
- Beginner-friendly Python OOP project

---

## 🛠️ Technologies Used

- Python 3.x
- Object-Oriented Programming (OOP)
